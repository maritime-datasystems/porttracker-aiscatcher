If You Want to Build Your Own Receiver/Aggregator
You can use the existing components to build a private aggregator:

bash
# Station 1 sends to your server
AIS-catcher -P your-server-ip 4242 JSON on
# Your server receives and stores
AIS-catcher -x 0.0.0.0 4242 -D "postgresql://user:pass@localhost/ais"
The pieces exist for you to build your own server-side, but the actual aiscatcher.org infrastructure is maintained privately by jvde-github.

The aiscatcher.org Server
The community hub server at aiscatcher.org is a separate, closed-source backend project. What we know:

Runs on 185.77.96.227:4242
Receives TCP connections with JSON data
Deduplicates messages from multiple stations
Provides the web interface at www.aiscatcher.org
The server-side processing, aggregation, and web API are maintained by the project author (jvde-github) separately and are not open source.
